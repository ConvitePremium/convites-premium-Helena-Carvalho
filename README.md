# convites-premium-Helena-Carvalho

Convite digital premium criado na Engine V2.

## Como publicar no GitHub Pages

1. Crie o repositório `convites-premium-Helena-Carvalho`.
2. Envie todos os arquivos deste pacote para a raiz do repositório.
3. Ative o GitHub Pages em **Settings > Pages**.
4. Selecione a branch principal e a pasta `/root`.

## Engine V2

- Mobile first.
- Imagens WebP otimizadas.
- Vídeo MP4 com fast start.
- Música otimizada com volume reduzido para 30%.
- Carregamento progressivo dos assets.
- Hotspots configuráveis em `config.js`.
- Modo de ajuste: adicione `?debug=1` ao final do link.

## Arquivos principais

- `index.html`: estrutura da engine.
- `config.js`: informações do convite e links.
- `assets/preview.webp`: capa para WhatsApp.
- `assets/principal.webp`: tela principal.
- `assets/presentes.webp`: tela de sugestões.
- `assets/video.mp4`: vídeo de abertura.
- `assets/musica.mp3`: música de fundo.
